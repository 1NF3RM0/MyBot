MyBot — Deriv Trading Bot

A fully asynchronous Python trading bot for the Deriv platform. It automates trading with a multi-strategy engine, real-time market classification, and adaptive parameter tuning.

Core Features

Dynamic Symbol Fetching

Automatically fetches tradable symbols from the Deriv API.

Advanced Technical Analysis

Uses RSI, MACD, Bollinger Bands, Ichimoku Cloud, ATR, and SMAs.

Multi-Strategy Confirmation

Executes trades only when two or more strategies agree, ensuring higher confidence signals.

Parameter Auto-Tuning

Dynamically adjusts risk, thresholds, and cooldowns based on market volatility.

Strategy Lifecycle Management

Enables/disables strategies based on their recent performance.

Contract Monitoring & Early Exit

Monitors open positions and exits early if conditions turn unfavorable (e.g., RSI overbought/oversold).

Robust Logging

Logs all actions to a local SQLite database for analysis and debugging.

Graceful Shutdown

Saves open trades and state on exit to ensure safe recovery.

Asynchronous Error Handling

Includes exponential backoff retries for robust API communication.

From Bot to Platform: The Roadmap

To evolve MyBot from a script into a polished, deployable web application, here is the development roadmap.

Build a web dashboard (React / Vue) to display performance, open trades, logs, and strategy analytics. Show real-time updates and include charts for profit, drawdowns, and other metrics.

Wrap the bot core into a web backend (FastAPI / Django) with REST endpoints for controlling the bot, fetching status, and adjusting parameters. Use WebSockets to push real-time updates.

Use a robust database like PostgreSQL to persist trade logs, strategy performance, and user data. Integrate a backtesting engine to simulate strategies on historical data.

Package the application with Docker for easy deployment on cloud providers (AWS, DigitalOcean). Implement CI/CD for automated testing and deployment, and use monitoring tools like Prometheus and Grafana.

Store API credentials securely using environment variables or a secrets manager. Validate all user inputs, add rate limiting to API endpoints, and ensure the bot can recover gracefully from failures.

Getting Started


# 1. Clone the repository
git clone https://github.com/your-username/MyBot.git
cd MyBot

# 2. Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure your API keys
# Edit src/config.py and add your Deriv App ID and API Token

# 5. Run the bot
python3 -m src.core

# 6. Run tests (optional)
python -m unittest discover tests
